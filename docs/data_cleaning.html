¿Por qué el MAE% de ventas es mayor (17.6% vs 10%)? Los precios de venta tienen más dispersión por barrio (Puerto Madero a $650K median vs Villa Lugano a $65K). Sin amenities para discriminar dentro de un barrio, el modelo tiene menos información para refinar la predicción. Con el dataset de ZonaProp 2026 (con amenities) el MAE% debería bajar significativamente.
